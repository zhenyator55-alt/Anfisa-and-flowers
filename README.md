# Гра «Анфіса і квіти» 🐱🌸

Гра **«Анфіса і квіти»** — це інтерактивний програмний продукт, розроблений у межах курсової роботи студентки Полтавського політехнічного фахового коледжу. Проєкт реалізує цікаву алгоритмічну та комбінаторну задачу, загорнуту в сюжетну лінію про дівчинку Анфісу та кота Мурзика. 

Гра розроблена мовою **C++** і підтримує два режими роботи: **консольний** та **графічний** (на базі бібліотеки SFML). Вона створена як для розваги, так і для розвитку логічного й математичного мислення у школярів.

## 📋 Суть задачі

Ключова задача гравця — проаналізувати умови та вивести математичну формулу:
cheese = 1 + (m - 1) * (n - 1)

---

## 🚀 Функціонал програми

* **Початкове меню:** дозволяє вибрати консольний режим (1), графічний режим (2) або вийти з гри (3).
* **Консольний інтерфейс:** * Повне пояснення умов задачі через діалоги головних героїв.
    * Система з 5 додаткових спроб для введення відповіді.
    * Автоматична підказка (виведення формули) від Мурзика, якщо гравець помилився 5 разів поспіль.
* **Графічний інтерфейс (GUI на SFML):**
    * Візуалізація сітки поля (максимальний розмір відображення 5x5 для збереження гармонії інтерфейсу, розмір клітинок — 60x60 пікселів).
    * Приємні пастельні кольори вікна та інтерактивне поле введення відповіді (Result).
    * Спрайти головних героїв — Анфіси та Мурзика.
    * Динамічні хмаринки з репліками-підказками або привітаннями залежно від результату.
    * Необмежена кількість спроб у графічному режимі.

## 📦 Встановлення та запуск

### Попередні вимоги
Для успішного запуску графічної версії у вашій системі має бути встановлена та налаштована бібліотека **SFML 3.0**.

### Інструкція з запуску:
1. Завантажте архів із сирцями або клонуйте репозиторій:
   git clone https://github.com/zhenyator55-alt/Anfisa-and-flowers.git

2. Розпакуйте архів та відкрийте папку kyrsova у вашому редакторі коду (наприклад, VS Code).
3. Скомпілюйте головний файл проєкту kyrsova.cpp за допомогою вашого C++ компилятора, підключивши прапорці SFML:
   g++ -c kyrsova.cpp
   g++ kyrsova.o -o anfisa-game -lsfml-graphics -lsfml-window -lsfml-system

## 🧪 Тестування

Продукт пройшов всебічне тестування за допомогою таких методологій:
* **Black Box (Чорна скринька):** перевірено поведінку меню, реакцію на некоректне введення символів, ліміти графічної сітки поля та механіку лічильника спроб.
* **White Box (Біла скринька):** протестовано внутрішню логіку обробки формули, зчитування подій клавіатури (Unicode введення чисел та обробка клавіші Enter/код 13).

Завдяки тестуванню підтверджено стабільну роботу та відсутність критичних багів.
