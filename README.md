# Гра «Анфіса і квіти» 🐱🌸

[cite_start]Гра **«Анфіса і квіти»** — це інтерактивний програмний продукт, розроблений у межах курсової роботи студентки Полтавського політехнічного фахового коледжу[cite: 1, 13, 167]. [cite_start]Проєкт реалізує цікаву алгоритмічну та комбінаторну задачу, загорнуту в сюжетну лінію про дівчинку Анфісу та кота Мурзика[cite: 14, 24, 49]. 

[cite_start]Гра розроблена мовою **C++** і підтримує два режими роботи: **консольний** та **графічний** (на базі бібліотеки SFML)[cite: 15]. [cite_start]Вона створена як для розваги, так і для розвитку логічного й математичного мислення у школярів[cite: 23, 174].

## 📋 Сюжет та суть задачі

[cite_start]За сюжетом, розумна дівчинка Анфіса отримує на олімпіаді завдання[cite: 24, 29]. [cite_start]Їй потрібно знайти формулу, яка виражає найбільшу кількість шматочків сиру, що вона може отримати залежно від розмірів заданого Мурзиком поля ($m \times n$)[cite: 30, 83]. [cite_start]При цьому Анфіса має зберегти якнайбільше квітів, але під час кожного переміщення по сітці вона зобов'язана зібрати щонайменше одну квітку[cite: 84].

[cite_start]Ключова задача гравця — проаналізувати умови та вивести математичну формулу[cite: 83]:
$$\text{cheese} = 1 + (m - 1) \times (n - 1)$$

## 🚀 Функціонал програми

* [cite_start]**Початкове меню:** дозволяє вибрати консольний режим (1), графічний режим (2) або вийти з гри (3)[cite: 78, 82].
* [cite_start]**Консольний інтерфейс:** * Повне пояснення умов задачі через діалоги головних героїв[cite: 77].
    * [cite_start]Система з 5 додаткових спроб для введення відповіді[cite: 87].
    * [cite_start]Автоматична підказка (виведення формули) від Мурзика, якщо гравець помилився 5 разів поспіль[cite: 88].
* **Графічний інтерфейс (GUI на SFML):**
    * [cite_start]Візуалізація сітки поля (максимальний розмір відображення $5 \times 5$ для збереження гармонії інтерфейсу, розмір клітинок — $60 \times 60$ пікселів)[cite: 96, 98].
    * [cite_start]Приємні пастельні кольори вікна та інтерактивне поле введення відповіді (`Result`)[cite: 64, 163].
    * [cite_start]Спрайти головних героїв — Анфіси та Мурзика[cite: 70].
    * [cite_start]Динамічні хмаринки з репліками-підказками або привітаннями залежно від результату[cite: 103, 104].
    * [cite_start]Необмежена кількість спроб у графічному режимі[cite: 165].
